# Política de Privacidade — Kowalski

**Última atualização:** 20 de março de 2026

## Resumo

Kowalski não coleta, armazena ou transmite nenhum dado pessoal do usuário. Todo o processamento acontece localmente no navegador.

## Dados coletados

Nenhum.

## Dados transmitidos

Nenhum. A extensão não faz requisições a servidores externos. Ela apenas lê o HTML da página do FNET/CVM que já está aberta no navegador e aplica transformações visuais localmente.

## Permissões

- **storage**: utilizada para salvar preferências locais do usuário (como o estado do toggle entre visão Kowalski e original). Os dados ficam armazenados apenas no navegador do usuário via `chrome.storage.local`.

## Contato

Para dúvidas sobre esta política, abra uma issue em https://github.com/geeksilva97/kowalski/issues.
